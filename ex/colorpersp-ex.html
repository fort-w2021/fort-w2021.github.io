Die Funktion persp() aus dem Basispaket graphics zeichnet 3D-Plots. Das animation-Paket kann animierte Bilder in vielen Formaten erzeugen.

  1. Lesen sie die Hilfe zu persp und führen Sie die Beispiele aus.

  2. Schreiben Sie eine Funktion
colorpersp <- function(x, y, z, z_palette) {
#TODO
}

die Beispiel 4 (“Surface colours corresponding to z-values”) aus der Hilfe von persp() direkt implementiert. z_palette ist dabei ein Vektor von Farben in der die Facetten der geplotteten Oberfläche je nach \(z\)-Wert eingefärbt werden sollen. Ihre Funktion sollte folgende Grafik reproduzieren können:

n_grid <- 50
x <- seq(-30, 10, length = n_grid)
y <- x
drip_drop <- function(x, y, time = pi / 2) {
  r <- sqrt(x^2 + y^2)
  sin(time) * sin(r) / r
}
z <- outer(x, y, drip_drop)
colors <- rainbow(100)
colorpersp(x = x, y = y, z = z, z_palette = colors)
